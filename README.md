# Telemetr Anti-Fake Bot (MVP)

## Что делает
Telegram-бот для оценки вероятности накрутки Telegram-канала на основе ER и просмотров через Telemetr API.

## Что нужно
- Python 3.10+
- Telegram Bot Token (BOT_TOKEN)
- Telemetr API Key (TELEMETR_API_KEY)

## Переменные окружения
- BOT_TOKEN
- TELEMETR_API_KEY

## Запуск
pip install -r requirements.txt
python main.py

## Использование
Отправь боту:
@channelname
или
https://t.me/channelname
